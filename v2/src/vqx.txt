
  Usage: vqx <command> ...

    vqx -h		    displays this help text
    vqx -v		    a bit of version info about vqx

    vqx eval 'code...'	    evaluate the specified Thrill code
    vqx lsk filename	    show contents of a starkit
    vqx views filename	    dump view structure of a Metakit datafile
    vqx view3 filename	    dump view structure of a v3 datafile
